## Assessment Title
HTML Fundamentals Assessment
TASK-1
**Home Page (index.html):**
The home page introduces the portfolio website with a hero section, navigation menu, and brief overview sections. It uses semantic HTML elements like `<header>`, `<nav>`, `<main>`, and `<section>` to organize content clearly.

**About Page (about.html):**
The about page presents a professional biography in paragraph form along with a structured skills table and testimonial blockquotes. It demonstrates use of tables with attributes like `border`, `bgcolor`, and `align` to display skills in a formatted way.

**Projects Page (projects.html):**
The projects page showcases at least six projects, each containing an image, title, description, and an external link that opens in a new tab. It is designed to highlight work samples in a structured and visually consistent format.

**Contact Page (contact.html):**
The contact page includes a form with labeled inputs for name, email, phone, subject selection, and message textarea. It allows users to submit inquiries and demonstrates proper use of form elements and accessibility with `<label>` tags.
